Width Property in CSS

While you can put text up in the header, once you have applied normal header style to a web page you will find that this text gets in the way and becomes difficult to read.

Three Basic Width Settings:

  • width
  • min-width
  • max-width

Those all do exactly what they imply. You can also set the height of an element, though it is more common to adjust the line-height property instead.

Units of Length in HTML

  • em - relative font size of parent element
  • rem - relative font size of the overall document
  • % - percent; this will be a percent of the containing element's baseline
  • pt - points; this is used selectively when coding CSS specifically for printer output of Web content; it should not be used anywhere else
  • px - pixels; you'll see this used a lot in tutorial Websites, but it is an archaic practice that you should avoid in your own code if at all possible

You can also use in (inches), cm (centimeters), and a variety of other equally bad choices for Web design; avoid them because we cannot control how they play out on different display types. Sometimes pixels are unavoidable so we use them, but these other units are completely avoidable so there is no excuse ever to use them.
